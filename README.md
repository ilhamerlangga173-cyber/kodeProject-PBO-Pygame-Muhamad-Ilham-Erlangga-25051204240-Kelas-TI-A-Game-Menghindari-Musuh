🎮 Game Menghindari Musuh (Pygame - PBO)
Proyek ini merupakan implementasi Pemrograman Berorientasi Objek (PBO) menggunakan bahasa pemrograman Python dengan library Pygame. Game ini adalah permainan sederhana di mana pemain harus menghindari musuh yang jatuh dari atas layar. Setiap musuh yang berhasil dihindari akan menambah skor. Permainan akan berakhir jika pemain bertabrakan dengan musuh.

⚙️ Fitur Utama
- Kontrol pemain menggunakan keyboard (kiri & kanan)
- Musuh muncul dari atas dan bergerak ke bawah secara acak
- Sistem deteksi tabrakan (collision detection)
- Menggunakan konsep Object-Oriented Programming (OOP)
- FPS (Frame Per Second) agar game stabil
- Tampilan Game Over

🧱 Konsep PBO yang Digunakan
- Class & Object (GameObject, Player, Enemy)
- Inheritance (Player & Enemy dari GameObject)
- Encapsulation (atribut dan method dalam class)

🛠️ Teknologi yang Digunakan
- Python 3
- Pygame

▶️ Cara Menjalankan Program
1. Install Python
2. Install pygame:
   pip install pygame
3. Jalankan program:
   tugas_game.py

🎯 Cara Bermain
- Gunakan tombol ← (kiri) dan → (kanan) untuk menggerakkan pemain
- Hindari semua musuh yang jatuh dari atas
- Skor akan bertambah setiap berhasil menghindari musuh
- Game berakhir jika terkena musuh
